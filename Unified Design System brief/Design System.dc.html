Casing rules
  • One label style. All tracked caps labels are exactly 10.5px / .14em / 600. The 8–11px, .06–.3em scatter is retired.
  • Serif is never uppercase — it names things (titles, beats, NPCs) in sentence case.
  • Buttons are sentence case, 12.5px / 600. Only the on-air family (TAKE, LIVE, ON AIR) shouts.
  • Gold text only means on air. Labels default muted; faint for hints; green for active.
Serif vs sans
Serif (IM Fell English) is the séance voice: window titles, section headers, in-fiction content, broadcast display. Sans is the operator voice: controls, labels, data. Numerals never use IM Fell (its figures wobble) — tabular Cormorant SemiBold replaces ad-hoc Georgia everywhere digits align: initiative, clocks, dice, session timer.
