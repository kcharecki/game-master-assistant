# Unified Design System brief

**Baseline for styling.** These files are the canonical reference for the app's
visual language (design tokens, components, Stage showcase).

## Do not modify

Do **not** edit any file in this folder unless explicitly prompted to do so.
Treat them as read-only source-of-truth:

- `Design System.dc.html` — component/token catalog
- `Stage Showcase.dc.html` — Stage module reference layout
- `image-slot.js`, `support.js` — supporting assets
- `.thumbnail`

When styling modules, match this baseline. Change the baseline only on explicit request.
